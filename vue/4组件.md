## vueComponent 

简称 VC

```javaScript

const school = vue.extend({
	name: "school",
	template: "<div>{{name}}</div>",
	data(){
		return{
			name: "学校名称"
		}
	}
})

const hello = vue.extend({
	name: "school",
	template: "<div>{{name}}</div>",
	data(){
		return{
			name: "学校名称"
		}
	}
})

// VM
new vue({
	components: {}
})
```

VueComponent 每次都返回一个全新的VueComponent

VM 管理的VC

VC 里面的this都是指自己 （VC）
VM 里面的this都是指自己（VM）
